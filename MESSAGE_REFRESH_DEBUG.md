# Message Refresh Debugging

## Check Browser Console

After refresh, you should see:
- Fetching user and messages for: X
- Messages received: [array]
- Number of messages: X
- Messages state changed, count: X

## If Messages Show in Console But Not on Screen

The backend is working, but rendering has an issue.

## Quick Test

Open browser console and refresh the page. Look for the logs above.

If you see messages in console but not on screen, let me know what the console shows.
